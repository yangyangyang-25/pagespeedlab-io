## Why this matters

When a Static Site Needs an Image CDN matters because small publishing mistakes can affect many pages on a
static content site. A clear workflow keeps the site useful for readers and easy
to maintain.

## Setup

Start with the target keyword: `image cdn static site`. Define the reader problem, the
page type, and the expected outcome before editing templates or publishing new
content.

## Implementation notes

Resize images to the largest display size they actually need. Compress assets before publishing and avoid using screenshots as full-width hero images by default. Write alt text that explains the image purpose instead of stuffing keywords.
